# sortes_vergilianae
Vigirlian Lottery - Returns a random line from Vigril's Aeneid
NOTE: If you want to run this from bin you need to modify the path to book in sortes_vergilianae.c Give absolute/path/to/aeneid.txt

INSTALLATION:
$ gcc -o sortes_vergilianae sortes_vergilianae.c

USAGE:
$ ./sortes_vergilianae
